📊 About This Dashboard

🏗️ Leicester Flood Structures

990 flood defence structures and features from Leicester City Council. Shows walls, culverts, and other flood protection infrastructure.

📅 Historic Flood Outlines

Historic Flood Map polygons showing maximum recorded flood extent. The source file does not include event dates or years, so this layer shows outlines rather than dated events.

🌊 EA Flood Zones

Environment Agency Flood Zones for planning.
Base layer: Standard flood zones without climate-change allowance.
Climate change layer: Flood zones with climate-change allowance.

🟩 County Wards

Ward boundaries shown as a separate overlay layer for reference. This layer helps compare flood risk features against local administrative areas.
